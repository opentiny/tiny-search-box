# Slots

| 名称           | 说明                                                                                              |
| -------------- | ------------------------------------------------------------------------------------------------- |
| suffix-icon    | 在帮助图标前面添加自定义图标按钮                                                                  |
| first-panel    | 自定义第一级面板内容，用于覆盖默认的筛选属性列表                                                  |
| second-panel   | 自定义第二级面板内容，用于覆盖默认的属性值选择面板                                                |
| {slotName}     | type=custom 时生效，自定义二级面板的插槽名，由 item.slotName 指定                                  |
| {slotName}-edit | type=custom 时生效，自定义编辑态面板的插槽名，由 item.slotName + '-edit' 指定                     |
